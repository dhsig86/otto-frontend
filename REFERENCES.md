## Observações de qualidade e uso

- **Priorizar fontes primárias** (diretrizes oficiais, revisões sistemáticas, capítulos acadêmicos). Recursos leigos (WebMD, blogs) e repositórios não revisados (alguns itens do ResearchGate) **não** devem ser usados como base de decisão clínica sem confirmação por fonte primária.  
- **Rastreamento no código:** cada regra em `rules_otorrino.json` e cada heurística de `diagnostics.js` deve conter um campo/nota com `ref: [ID curto]` apontando para um item desta lista.  
- **Controle de versão:** quando um guideline for atualizado, criar novo item com ano e apontar as regras revisadas.

### Exemplo de citação no código
```json
{
  "name": "Vertigem posicional paroxística benigna",
  "prior": 0.30,
  "notes": "ref: [AAO‑HNSF BPPV 2017]; considerar fatores de recorrência – [J Clin Med 2021 BPPV Recurrence]"
}
```
